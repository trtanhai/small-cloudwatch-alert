# AWS EC2 Monitoring & Alerting (CloudWatch + SNS)

This project demonstrates how to monitor an EC2 instance using CloudWatch Agent, create custom metrics (CPU, RAM, Disk), and configure an SNS alerting system.  
This project is 100% Free Tier compatible.
